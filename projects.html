Lyric finder site

As part of my Advanced Higher Computing Science course a student had to create a web application that had a front and back end. I chose to create a site that allowed a user to enter text. The website would then take the users input and compare it to a database filled with lyrics. If the user inputed data matched part of a lyric it would then show the user the song, the album and the artist their lyric was from. I used HTML, CSS, PHP and MySQL to create this website. Below is a link to a GitHub Repo containing the code for this site. Feel free to check it out!
